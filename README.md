# CodeAlpha-Stock-Portfolio-Tracker
Task 2 for the CodeAlpha Python Programming Internship: A simple stock portfolio tracker that calculates total investment value based on user inputs and hardcoded prices.

## Description
A simple terminal-based Stock Portfolio Tracker developed as Task 2 for the CodeAlpha Python Programming Internship. This application allows users to build a custom mock portfolio by entering stock names and quantities, automatically calculating the total value based on predefined stock data.

## Features & Scope
* **User Input:** Accepts stock ticker names and the respective quantity owned from the user.
* **Predefined Price Dictionary:** References a hardcoded dictionary mapping stock symbols to their current market prices (e.g., `{"AAPL": 180, ...}`).
* **Portfolio Evaluation:** Displays the total real-time value of the user's combined investment.
* **Data Export (Optional):** Saves the final calculated summary into a text file (`.txt`) or a spreadsheet data file.

## Key Concepts & Technologies Used
* **Language:** Python
* **Concepts:** `dictionary`, console `input/output`, basic arithmetic operations, and optional `file handling`.
